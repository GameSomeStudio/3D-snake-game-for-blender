# 🐍 Snake Game 3D for BLENDER
## Play Classic Snake in Your 3D Viewport

Take a fun break during long render sessions! This addon brings the classic Snake arcade game directly into 3D Viewport. 
Control a growing snake (with Suzanne as the head!) to collect spinning golden coins while avoiding walls and your own tail.

- Fully playable inside 3D Viewport - no external windows needed
- Dynamic snake that grows with each coin collected
- Arrow key controls 
- Pause (P) and restart (Enter) support
- Score display with game over screen
- Adjustable camera angle
- Optional sound effects (procedural, no external files)
- Optional scene obstacles mode
- Graphics quality setting (HIGH/LOW) for performance
- Complete cleanup on exit - no leftover objects
